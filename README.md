# To-Do List

A small browser-based to-do app built with plain HTML, CSS, and JavaScript.
Tasks and theme preference are saved in `localStorage`, so the app keeps your
list between browser sessions without needing a backend.

## Live Demo

View the deployed app on GitHub Pages: https://appsolotech.github.io/to-do/

## Features

- Add, complete, edit, and delete tasks.
- Filter tasks by all, active, or completed.
- Clear all completed tasks at once.
- Track the number of active tasks remaining.
- Toggle between light and dark mode.
- Responsive layout for desktop and mobile screens.

## Project Files

- `index.html` - App markup and page structure.
- `styles.css` - Layout, theme, and responsive styles.
- `app.js` - Task state, local storage, filtering, editing, and event handling.

## Run Locally

No install step is required. This app has no package dependencies.

1. Open the project folder.
2. Open `index.html` in a web browser.

You can also run it with a lightweight local server from the project root:

```bash
explorer.exe index.html
```
